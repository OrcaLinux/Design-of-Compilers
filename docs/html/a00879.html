Detailed Description

Manages the scanning process for TINY language source code.

The TokenStreamBuilder class orchestrates the tokenization process. It uses the Scanner class to generate tokens and appends them to a vector of tokens for further processing.

Constructs a TokenStreamBuilder object with the given Scanner object.

This constructor initializes the TokenStreamBuilder with a reference to a Scanner object, which is used to generate tokens from the input source code.

Parameters
